## 授课语言核验（2026-06-14 更新）

毛豆不懂日语/韩语，因此「授课语言＝英文」是核心筛选项。已逐一核验 17 个项目的授课语言并在卡片/详情页标注：

- 🇬🇧 **英文授课 · 15 项**：UWC ISAK、Camp Asia、UWCSEA、Stamford 以外的国际学校/IB/英式学校 + ESL（QQ English）+ Niseko 英语户外营等，均为全英文环境。
- 🌐 **英文+ · 1 项**：Stamford American（默认英文授课，另设中文/西班牙语沉浸班可选）。
- 🇯🇵 **日语为主 · 1 项**：富士山下 Whole Earth 自然体验营（日本本土自然学校，官网仅日文、营会以日语进行）——保留展示但明确标注，点「🇬🇧 仅英文授课」筛选会被过滤掉。

点顶部「🇬🇧 仅英文授课」筛选芯片即可只看 16 个英文/英文+ 项目。每个项目详情页含「🗣️ 授课语言」一栏 + 来源核验链接 + 置信度。

2026-05-13 更新：按"不接受纯日语"口径，移除北海道纯日语项目，仅保留北海道英语/国际化项目；新增泰国非曼谷/普吉/清迈城市英文项目 5 个（Chonburi/Pattaya、Rayong、Hua Hin、Krabi/Railay）。
